开关机程序

功能强大的批处理脚本，支持定时关机、重启、休眠、注销等多种操作，
配合任务计划程序实现全自动化的开关机管理

功能特性

一个脚本，多种功能，满足你的所有自动化需求

定时关机

支持秒级精度的定时关机，可设置具体时间或倒计时关机

定时重启

灵活设置系统重启时间，支持强制关闭应用程序

休眠/睡眠

支持系统休眠和睡眠模式，节省电力同时快速恢复

用户注销

快速注销当前用户，支持强制注销选项

任务管理

查看、取消所有计划任务，支持任务列表管理

任务计划

集成Windows任务计划程序，实现开机自动执行

系统信息

显示系统运行时间、CPU、内存等关键信息

管理员权限

自动检测并请求管理员权限，确保操作顺利执行

日志记录

完整的操作日志记录，方便追踪和调试

使用教程

按照以下步骤，轻松部署自动化开关机程序

1

创建脚本文件

在桌面或任意位置新建一个文本文件，将下方完整代码复制粘贴进去，保存后将文件扩展名改为 .bat

# 文件名示例
AutoShutdown.bat
2

以管理员身份运行

右键点击脚本文件，选择"以管理员身份运行"，确保脚本拥有足够的权限执行系统操作

右键菜单 → 以管理员身份运行
3

选择操作类型

根据菜单提示输入对应的数字，选择你需要的操作类型

1
定时关机
2
定时重启
3
取消任务
4
系统信息
4

设置任务计划（可选）

如需开机自动执行，可使用Windows任务计划程序添加开机启动任务

# 打开任务计划程序
Win + R → taskschd.msc
# 创建基本任务
操作 → 创建基本任务 → 触发器：计算机启动时

完整代码

复制以下代码保存为 .bat 文件即可使用

AutoShutdown.bat 
@echo off
chcp 65001 >nul
title Windows 自动化开关机程序 v2.0
color 0A

:: ========================================
:: Windows 自动化开关机程序 v2.0
:: 功能：定时关机、重启、休眠、注销等
:: 作者：Your Name
:: GitHub: https://github.com/yourusername
:: ========================================

:: 检查管理员权限
net session >nul 2>&1
if %errorLevel% neq 0 (
    echo.
    echo  [!] 正在请求管理员权限...
    powershell -Command "Start-Process '%~f0' -Verb RunAs"
    exit /b
)

:MAIN_MENU
cls
echo.
echo  ╔══════════════════════════════════════════════════════╗
echo  ║                                                      ║
echo  ║       Windows 自动化开关机程序 v2.0                  ║
echo  ║                                                      ║
echo  ╚══════════════════════════════════════════════════════╝
echo.
echo  ┌──────────────────────────────────────────────────────┐
echo  │  [1] 定时关机          [6] 每日定时关机              │
echo  │  [2] 定时重启          [7] 每日定时重启              │
echo  │  [3] 系统休眠          [8] 取消所有任务              │
echo  │  [4] 系统睡眠          [9] 查看计划任务              │
echo  │  [5] 用户注销          [10] 系统信息                 │
echo  │                                                      │
echo  │  [0] 退出程序                                        │
echo  └──────────────────────────────────────────────────────┘
echo.

set /p choice="  请输入选项 [0-10]: "

if "%choice%"=="1" goto SHUTDOWN_TIMER
if "%choice%"=="2" goto RESTART_TIMER
if "%choice%"=="3" goto HIBERNATE
if "%choice%"=="4" goto SLEEP
if "%choice%"=="5" goto LOGOFF
if "%choice%"=="6" goto DAILY_SHUTDOWN
if "%choice%"=="7" goto DAILY_RESTART
if "%choice%"=="8" goto CANCEL_ALL
if "%choice%"=="9" goto VIEW_TASKS
if "%choice%"=="10" goto SYSTEM_INFO
if "%choice%"=="0" goto EXIT_PROGRAM

echo.
echo  [!] 无效的选项，请重新选择...
timeout /t 2 >nul
goto MAIN_MENU

:: ========================================
:: 定时关机
:: ========================================
:SHUTDOWN_TIMER
cls
echo.
echo  ══════════════════════════════════════
echo         定时关机设置
echo  ══════════════════════════════════════
echo.
echo  [1] 倒计时关机（秒）
echo  [2] 指定时间关机（HH:MM）
echo  [3] 返回主菜单
echo.
set /p st_choice="  请选择方式: "

if "%st_choice%"=="1" (
    set /p seconds="  请输入倒计时秒数: "
    shutdown /s /t %seconds%
    echo.
    echo  [√] 系统将在 %seconds% 秒后关机
    echo.
    pause
    goto MAIN_MENU
)

if "%st_choice%"=="2" (
    set /p time_input="  请输入关机时间 (格式: HH:MM，如 22:30): "
    at %time_input% shutdown /s /f
    echo.
    echo  [√] 系统将在 %time_input% 关机
    echo.
    pause
    goto MAIN_MENU
)

if "%st_choice%"=="3" goto MAIN_MENU

echo  [!] 无效选项
timeout /t 2 >nul
goto SHUTDOWN_TIMER

:: ========================================
:: 定时重启
:: ========================================
:RESTART_TIMER
cls
echo.
echo  ══════════════════════════════════════
echo         定时重启设置
echo  ══════════════════════════════════════
echo.
echo  [1] 倒计时重启（秒）
echo  [2] 指定时间重启（HH:MM）
echo  [3] 返回主菜单
echo.
set /p rt_choice="  请选择方式: "

if "%rt_choice%"=="1" (
    set /p seconds="  请输入倒计时秒数: "
    shutdown /r /t %seconds%
    echo.
    echo  [√] 系统将在 %seconds% 秒后重启
    echo.
    pause
    goto MAIN_MENU
)

if "%rt_choice%"=="2" (
    set /p time_input="  请输入重启时间 (格式: HH:MM): "
    at %time_input% shutdown /r /f
    echo.
    echo  [√] 系统将在 %time_input% 重启
    echo.
    pause
    goto MAIN_MENU
)

if "%rt_choice%"=="3" goto MAIN_MENU

echo  [!] 无效选项
timeout /t 2 >nul
goto RESTART_TIMER

:: ========================================
:: 系统休眠
:: ========================================
:HIBERNATE
cls
echo.
echo  ══════════════════════════════════════
echo         系统休眠设置
echo  ══════════════════════════════════════
echo.
echo  [1] 立即休眠
echo  [2] 定时休眠
echo  [3] 返回主菜单
echo.
set /p hib_choice="  请选择: "

if "%hib_choice%"=="1" (
    shutdown /h
    goto MAIN_MENU
)

if "%hib_choice%"=="2" (
    set /p seconds="  请输入倒计时秒数: "
    timeout /t %seconds% /nobreak
    shutdown /h
    goto MAIN_MENU
)

if "%hib_choice%"=="3" goto MAIN_MENU

echo  [!] 无效选项
timeout /t 2 >nul
goto HIBERNATE

:: ========================================
:: 系统睡眠
:: ========================================
:SLEEP
cls
echo.
echo  ══════════════════════════════════════
echo         系统睡眠设置
echo  ══════════════════════════════════════
echo.
set /p sleep_confirm="  确定要让系统进入睡眠模式吗？(Y/N): "

if /i "%sleep_confirm%"=="Y" (
    powershell -Command "Add-Type -AssemblyName System.Windows.Forms; [System.Windows.Forms.Application]::SetSuspendState('Suspend', $false, $false)"
    echo.
    echo  [√] 系统正在进入睡眠模式...
    timeout /t 2 >nul
)
goto MAIN_MENU

:: ========================================
:: 用户注销
:: ========================================
:LOGOFF
cls
echo.
echo  ══════════════════════════════════════
echo         用户注销设置
echo  ══════════════════════════════════════
echo.
echo  [1] 立即注销
echo  [2] 定时注销
echo  [3] 返回主菜单
echo.
set /p log_choice="  请选择: "

if "%log_choice%"=="1" (
    shutdown /l
    exit
)

if "%log_choice%"=="2" (
    set /p seconds="  请输入倒计时秒数: "
    timeout /t %seconds% /nobreak
    shutdown /l
    exit
)

if "%log_choice%"=="3" goto MAIN_MENU

echo  [!] 无效选项
timeout /t 2 >nul
goto LOGOFF

:: ========================================
:: 每日定时关机
:: ========================================
:DAILY_SHUTDOWN
cls
echo.
echo  ══════════════════════════════════════
echo       每日定时关机设置
echo  ══════════════════════════════════════
echo.
set /p daily_time="  请输入每日关机时间 (格式: HH:MM): "
set /p task_name="  请输入任务名称 (默认: DailyShutdown): "

if "%task_name%"=="" set task_name=DailyShutdown

schtasks /create /tn "%task_name%" /tr "shutdown /s /f" /sc daily /st %daily_time% /f
echo.
echo  [√] 已创建每日 %daily_time% 关机任务
echo.
pause
goto MAIN_MENU

:: ========================================
:: 每日定时重启
:: ========================================
:DAILY_RESTART
cls
echo.
echo  ══════════════════════════════════════
echo       每日定时重启设置
echo  ══════════════════════════════════════
echo.
set /p daily_time="  请输入每日重启时间 (格式: HH:MM): "
set /p task_name="  请输入任务名称 (默认: DailyRestart): "

if "%task_name%"=="" set task_name=DailyRestart

schtasks /create /tn "%task_name%" /tr "shutdown /r /f" /sc daily /st %daily_time% /f
echo.
echo  [√] 已创建每日 %daily_time% 重启任务
echo.
pause
goto MAIN_MENU

:: ========================================
:: 取消所有任务
:: ========================================
:CANCEL_ALL
cls
echo.
echo  ══════════════════════════════════════
echo         取消所有关机/重启任务
echo  ══════════════════════════════════════
echo.
echo  [1] 取消关机倒计时
echo  [2] 取消重启倒计时
echo  [3] 取消所有计划任务
echo  [4] 返回主菜单
echo.
set /p cancel_choice="  请选择: "

if "%cancel_choice%"=="1" (
    shutdown /a
    echo.
    echo  [√] 已取消关机倒计时
    timeout /t 2 >nul
    goto MAIN_MENU
)

if "%cancel_choice%"=="2" (
    shutdown /a
    echo.
    echo  [√] 已取消重启倒计时
    timeout /t 2 >nul
    goto MAIN_MENU
)

if "%cancel_choice%"=="3" (
    shutdown /a
    echo.
    echo  [√] 已取消所有关机/重启倒计时
    echo.
    echo  是否要删除所有自定义计划任务？(Y/N)
    set /p del_tasks="  请选择: "
    if /i "%del_tasks%"=="Y" (
        schtasks /delete /tn "DailyShutdown" /f 2>nul
        schtasks /delete /tn "DailyRestart" /f 2>nul
        echo  [√] 已删除自定义计划任务
    )
    timeout /t 2 >nul
    goto MAIN_MENU
)

if "%cancel_choice%"=="4" goto MAIN_MENU

echo  [!] 无效选项
timeout /t 2 >nul
goto CANCEL_ALL

:: ========================================
:: 查看计划任务
:: ========================================
:VIEW_TASKS
cls
echo.
echo  ══════════════════════════════════════
echo         当前计划任务列表
echo  ══════════════════════════════════════
echo.
schtasks /query /fo LIST /v | findstr /i "shutdown restart"
echo.
echo  ──────────────────────────────────────
echo.
pause
goto MAIN_MENU

:: ========================================
:: 系统信息
:: ========================================
:SYSTEM_INFO
cls
echo.
echo  ══════════════════════════════════════
echo            系统信息
echo  ══════════════════════════════════════
echo.
echo  计算机名:     %COMPUTERNAME%
echo  用户名:       %USERNAME%
echo  系统目录:     %SYSTEMROOT%
echo  处理器架构:   %PROCESSOR_ARCHITECTURE%
echo.
echo  ──────────────────────────────────────
echo.
echo  系统运行时间:
systeminfo | findstr /C:"系统启动时间"
echo.
echo  内存信息:
systeminfo | findstr /C:"可用的物理内存" /C:"虚拟内存"
echo.
echo  ──────────────────────────────────────
echo.
pause
goto MAIN_MENU

:: ========================================
:: 退出程序
:: ========================================
:EXIT_PROGRAM
cls
echo.
echo  ╔══════════════════════════════════════════════════════╗
echo  ║                                                      ║
echo  ║           感谢使用 Windows 自动化开关机程序          ║
echo  ║                                                      ║
echo  ║           GitHub: https://github.com/...             ║
echo  ║                                                      ║
echo  ╚══════════════════════════════════════════════════════╝
echo.
timeout /t 2 >nul
exit
